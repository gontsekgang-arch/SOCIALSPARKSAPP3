## Description

Social sparks is a simple Android application designed to stay socially conncted through out the day.The app allows uses to enter a time of day,such as morning,
  afternoon or night,and then provides a suitable social interation suggestion

  ## How to use
  The app is easy to use and user friendly.The user starts by entering time of the day into the input field.The app then processes this input using conditional(if)
  statements  to determine the correct suggestion.Once the input is checked,a matching social activity is displayed on the screen.If the user wants to try  a different time if,
  they can use the reset button to clear the input and start again.
